# Zork — The Great Underground Empire

A browser-based fan tribute to the classic 1981 Infocom text adventure.

## Play

Open `zork.html` in any modern browser. No server required.

## Features

- 30+ rooms spanning the white house, forest, underground empire, maze, and more
- Full text parser: `TAKE`, `DROP`, `EXAMINE`, `OPEN`, `READ`, `ATTACK`, `PUT IN`, compass movement
- Light/dark mechanics — wander into darkness and a grue will find you
- 8 treasures to collect and deposit in the trophy case for scoring
- Troll combat, locked trapdoors, a jeweled egg with a hidden canary inside
- Command history (↑↓ arrow keys)

## Commands

| Command | Description |
|---|---|
| `LOOK` / `L` | Describe your surroundings |
| `N` / `S` / `E` / `W` / `U` / `D` | Move in a direction |
| `TAKE <item>` / `TAKE ALL` | Pick up items |
| `DROP <item>` | Drop an item |
| `INVENTORY` / `I` | List carried items |
| `EXAMINE <item>` / `X` | Examine something |
| `READ <item>` | Read an item |
| `OPEN` / `CLOSE <thing>` | Open or close things |
| `LIGHT <lamp/torch>` | Toggle a light source |
| `PUT <item> IN <container>` | Place item in container |
| `ATTACK <enemy>` | Fight an enemy |
| `SCORE` | Show your score |
| `RESTART` | Start over |
